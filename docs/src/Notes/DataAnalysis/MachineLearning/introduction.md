# 机器学习基础

## 学习目标
- 理解机器学习基本概念和流程
- 掌握监督学习和无监督学习算法
- 能够进行模型评估和调优
- 了解特征工程方法

## 核心知识点
- 机器学习工作流程
- 监督学习算法（线性回归、决策树等）
- 无监督学习（聚类、降维）
- 模型评估指标
- 过拟合与欠拟合
- 特征工程
- 模型部署基础
- 常见机器学习库（scikit-learn）
