# SQL 数据分析

## 学习目标
- 掌握 SQL 基础查询语法
- 能够进行复杂的数据聚合和连接操作
- 理解数据库设计和优化
- 掌握窗口函数等高级特性

## 核心知识点
- SELECT 基础查询
- 表连接（JOIN）
- 聚合函数与 GROUP BY
- 子查询与 CTE
- 窗口函数
- 索引与性能优化
- 事务与锁
- 数据库设计范式

## 学习资源推荐
- 《SQL 必知必会》
- LeetCode 数据库题目
- SQL Zoo 互动教程
- Mode Analytics SQL 教程
