# Python 基础

## 学习目标
- 掌握 Python 基础语法和数据结构
- 理解函数式编程和面向对象编程
- 熟悉 Python 标准库的常用模块
- 能够使用 Python 进行基础数据处理

## 核心知识点
- 基础语法：变量、运算符、控制流
- 数据结构：列表、元组、字典、集合
- 函数：定义、参数传递、作用域、装饰器
- 面向对象编程：类、继承、多态
- 异常处理
- 文件操作
- 常用标准库：os, sys, datetime, json 等
